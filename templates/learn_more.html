Learn More

This website was created to conduct an analysis on the occurrences of crime in the GTA. The data was retrieved by the Toronto Police. The goal is to identify neighbourhoods which have a high crime rate, which crimes are repeated offenders, see how different burroughs in Toronto differ in their crimes and to create a model which given the type and time of crime can accurately predict the location of the crime. The overall results of this analysis can be found in the main page of the website. Upon creating an account and logging in you will also be given access to the model which can predict the location of the crime given the necessary information. If you would like to support the prevention of crime in Canada, you can refer to the link below which will direct you to the Government of Canada's official page regarding crime and crime prevention. If you have any questions, feel free to reach out to me using the contact form on the main page.
